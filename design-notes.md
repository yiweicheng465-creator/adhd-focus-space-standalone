# Design Reference Analysis

## VJ Type (vj-type.com/collections/cako)
- **Background**: warm off-white/parchment (#E8E4D8 equivalent)
- **Buttons**: 
  - "TRIALS" = outlined, thin 1px border, all-caps, small tracking, no fill, square corners (no radius)
  - "BUY" = solid black fill, white all-caps text, square corners, compact padding
  - Both use VERY small uppercase tracking-widest text, no rounded corners
  - Hover: subtle invert or underline — not a color fill change
- **Typography**: 
  - All nav = tiny all-caps tracking-widest, very light weight
  - Headings = large display serif, high contrast
  - No rounded corners anywhere
- **Spacing**: generous whitespace, asymmetric
- **Color**: near-monochrome — black on cream, with accent colors only in illustrations

## Little Valley Studio (littlevalleystudio.com)
- **Background**: warm sage-green/cream split
- **Logo**: "LITTLE *Valley* STUDIO" — mixes serif italic with small-caps, very refined
- **Buttons**:
  - "TAKE ME THERE" = all-caps, small tracking, thin underline or border, minimal
  - Navigation = all-caps, very small, tracking-wide, no decoration
- **Typography**:
  - Section labels = "PROJECT IDEAS" in tiny all-caps with a short rule below
  - Headings = large elegant serif (similar to Playfair)
  - Body = small, light, centered
- **Color**: warm sage green + cream + terracotta tones — very Morandi
- **Details**: thin horizontal rules as section dividers, no rounded corners

## Key Button Design Principles to Apply:
1. **NO rounded corners** — square or very subtly rounded (2px max)
2. **All-caps + tracking-widest** for button text
3. **Two button types**:
   - PRIMARY: solid dark fill (griffin charcoal or coral), white text, square
   - SECONDARY/GHOST: transparent, 1px thin border, dark text, square
4. **Thin underline hover** on ghost buttons instead of fill change
5. **Font**: DM Sans or similar — NOT Inter, NOT rounded
6. **Size**: compact — not large pill-shaped buttons
7. **No shadows** on buttons — flat, editorial
